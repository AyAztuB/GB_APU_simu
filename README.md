# GB_APU_simu
Some scripts used to demonstrate how the GameBoy APU works

## Dependencies

- python3
- tkinter
- matplotlib
- numpy
- sounddevice
- soundfile

### Debian linux packages

```sh
sudo apt install python3-tk portaudio19-dev
python3 -m venv .venv
.venv/bin/pip3 install numpy matplotlib sounddevice soundfile
```
